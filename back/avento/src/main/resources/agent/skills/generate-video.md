# Gera ou anima um vídeo curto local pelo ComfyUI
Gatilhos: gera video, gerar video, gere video, cria video, criar video, crie video, gera um video, cria um video, crie um video, anima essa imagem, anima a imagem, animar imagem, transforma em video, image to video, text to video
Ferramenta: generate_video

Use generate_video com o argumento do usuário como prompt principal de movimento.

Regras:
- Use mode auto por padrão para animar a mídia mais recente do chat quando houver uma.
- Use mode image quando a referência for obrigatória e mode text somente quando o vídeo deve nascer do zero.
- Descreva ação, movimento de câmera e ritmo; não altere desnecessariamente a aparência da referência.
- Respeite duração e tamanho pedidos dentro dos limites da ferramenta.
- Não simule progresso. Use o job assíncrono real e deixe o chat mostrar status e estimativa.
