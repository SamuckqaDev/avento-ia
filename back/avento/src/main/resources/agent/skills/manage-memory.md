# Guarda, consulta ou remove contexto na memória persistente local
Gatilhos: guarde na memoria, guardar memoria, lembre disso, memorize isso, consultar memoria, buscar memoria, o que voce lembra

Procedimento:
1. Para consultar, use search_nodes quando houver um termo específico; use read_graph somente quando uma visão ampla for necessária.
2. Para abrir resultados conhecidos, use open_nodes.
3. Para gravar, alterar relações ou apagar memória, confirme que o servidor memory está conectado com list_mcp_servers; conecte-o quando necessário e use somente as ferramentas descobertas.
4. Guarde fatos estáveis e úteis, não transcrições inteiras nem segredos.
5. Confirme apenas o que a ferramenta persistiu ou removeu de verdade.

Não invente lembranças e não trate o histórico comum do chat como memória persistente.
