# Lê e analisa documentos locais usando extração real de conteúdo
Gatilhos: ler pdf, leia pdf, analisar pdf, analise pdf, resumir pdf, ler documento, analisar documento, resumir documento

Procedimento:
1. Use read_document com o caminho autorizado do PDF, Word, Excel, PowerPoint, EPUB, ZIP, imagem com OCR, áudio ou arquivo textual.
2. Baseie a resposta somente no conteúdo retornado pela ferramenta.
3. Para resumo, preserve decisões, números, datas e pendências importantes.
4. Para perguntas, cite a seção ou trecho identificável do documento quando disponível.

Não use read_file para binários e não invente conteúdo quando a extração vier vazia ou falhar.
