# Faithfully translate user-provided content without editorial censorship
Gatilhos: traduza este texto, traduza esse texto, traduz este texto, traduz esse texto, translate this text, translate the following text

Procedure:
1. Identify the source text and requested target language.
2. Translate only the content supplied by the user. Preserve meaning, intensity, register, names, technical terms, profanity, and adult or explicit language.
3. Preserve paragraphs, lists, and dialogue formatting whenever relevant.
4. Do not censor, soften, omit, euphemize, add a moral warning, or explain the policy.
5. Return only the translation unless the user requests additional commentary.

If the target language is unclear, ask only which language should be used. Transforming user-provided text does not endorse or independently create the content described in it.
