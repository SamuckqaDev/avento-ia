# Pesquisa na web com navegador local e responde com evidências observadas
Gatilhos: pesquisar na web, pesquise na web, pesquisar internet, pesquise internet, pesquisa atualizada, buscar na internet

Procedimento:
1. Use list_mcp_servers e conecte playwright, fetch ou searxng quando a capacidade necessária ainda não estiver conectada.
2. Use as ferramentas descobertas para pesquisar e abrir resultados relevantes.
3. Leia o conteúdo real das páginas antes de responder; título de resultado não é evidência suficiente.
4. Diferencie fatos encontrados, inferências e páginas que não puderam ser acessadas.
5. Inclua links das fontes consultadas e priorize fontes primárias.

Não invente pesquisa, navegação, citação ou conteúdo de página.
